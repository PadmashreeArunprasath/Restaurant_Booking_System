**Restaurant Booking System**

*Overview*
    
 This is a simple, non-responsive web application built using HTML and CSS to create a visually appealing interface for a restaurant booking system. It includes pages for home, dine-in, delivery, and menu.

*Features*
 * Home Page: Welcoming page with a brief introduction to the restaurant.
 * Dine-In Page: Allows users to book a table for dine-in service, including date, time, and number of guests.
 * Delivery Page: Enables users to place delivery orders, specifying address and contact information.
 * Menu Page: Displays the restaurant's menu with item descriptions and prices.

*How to Use*
 * Open the Index File:
   Double-click on the index.html file to open it in your web browser.
   **ensure you have all the provides folders and files in a same folder.**
 * Navigate Through Pages:
   Use the navigation links provided on the home page to access the other pages.

*Customization*
To customize the appearance and functionality of the system:
 * Modify HTML:
   Edit the HTML files to change the content, layout, and structure.
 * Edit CSS:
   Modify the CSS files to adjust the styling, colors, and fonts.

*Note:*
As this is a non-responsive design, it may not display optimally on different screen sizes. For a more flexible solution, consider using CSS media queries or a responsive framework.

 * Responsive Design: Adapt the layout to different screen sizes.
 * Interactive Elements: Add interactive features like form validation and dynamic content.
